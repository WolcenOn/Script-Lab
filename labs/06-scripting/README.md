# Lab 06 - Scripting

## Objetivo

Convertir comandos manuales en una automatización sencilla y documentada.

---

## Relación con la guía

Consulta antes:

- Fundamentos de scripting.
- Automatización profesional.
- Integración con Script-Lab.

---

## Escenario

Ya sabes ejecutar comandos manuales para revisar un sistema. Ahora debes agruparlos en un pequeño flujo automatizado que genere una salida clara.

---

## Tareas

### Tarea 1: elegir objetivo

Elige una tarea:

- Inventario del sistema.
- Revisión de disco.
- Revisión de servicios.
- Diagnóstico de red.
- Informe de mantenimiento.

### Tarea 2: escribir pasos manuales

Antes de automatizar, escribe los comandos o acciones manuales.

### Tarea 3: crear versión inicial

Crea una primera versión en uno de estos lenguajes:

- Bash.
- PowerShell.
- Python.

### Tarea 4: añadir documentación

Incluye:

- Objetivo.
- Requisitos.
- Ejemplo de uso.
- Salida esperada.

### Tarea 5: mejorar

Añade al menos una mejora:

- Parámetros.
- Validación.
- Registro de ejecución.
- Salida a archivo.
- Modo resumen.


---

## Preguntas

1. ¿Qué problema resuelve tu automatización?
2. ¿Qué entradas necesita?
3. ¿Qué salida produce?
4. ¿Qué errores podrían ocurrir?
5. ¿Cómo comprobarías que funciona?
